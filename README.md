# complex_systems
Complex systems: River flooding dynamics

# How to install
## Python dependencies:
view pyproject.toml

## C++ compiling:
By far easiest on linux:
    
    apt install build-essential
    cd src/cpp_modules
    make

INSTRUCTIONS RUNNING EXPERIMENTS:
